# Pixar Business Performance & Artistic Trends Dashboard

## Overview

This project uses Power BI to analyze Pixar films’ business performance and artistic trends from 1995 to 2024, focusing on revenue, budget, ROI, profit, ratings, awards, genres, and performance trends.

## Tools

Power BI, Power Query, DAX, Data Modeling

## Key Metrics

* Total Revenue: $17.04B
* Total Budget: $4.50B
* Net Profit: $12.54B
* ROI: 278.64%

## What I Did

* Cleaned and transformed multiple tables using Power Query.
* Built a dim/fact data model for film, box office, public response, academy awards, genres, and crew data.
* Created DAX measures for Total Revenue, Total Budget, Net Profit, ROI, Cumulative Revenue, Oscar Wins, and review scores.
* Designed Power BI dashboards to analyze financial performance, audience/critic reception, and long-term business trends.

## Business Insights

The analysis suggests that Pixar’s financial performance depends heavily on global franchise success, international revenue, and careful budget control for non-franchise films.
